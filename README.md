```markdown
# 项目简介

## 核心功能
- 用户模块：
  - 用户注册与登录
  - 安全风控与登录凭证颁发
  - 个人信息管理
  - 收货地址管理
  - 账户安全设置
  - 消息通知
- 订单模块：
  - 交易订单生成与状态流
  - 订单查询与操作
- 收藏模块：
  - 商品收藏功能
  - 商品分类收藏
  - 评论系统
- 管理员模块：
  - 用户管理
  - 资源审核与操作
  - 资源管理
  - 评论监控
  - 推荐功能
  - 系统公告发布
- 额外功能：
  - 交易状态看板
  - 评论管理

## 适用场景
- 管理水果商城用户信息与交易
- 审核商品资源，确保内容合规
- 监控评论并维护社区氛围
- 根据用户行为进行商品推荐
- 发布系统公告与通知

## 技术特点
- 基于 SpringBoot 框架，提供高效稳定的后台管理
- 支持多用户角色，区分管理员与普通用户权限
- 集成安全验证与风控机制，保障用户信息安全
- 动态推荐算法，提升用户购物体验
- 富文本编辑器，方便发布系统公告
```
